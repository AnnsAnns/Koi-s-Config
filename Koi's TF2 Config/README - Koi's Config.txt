Installation process:

1. Copy the "koi_cfg" folder and the "autoexec.cfg" file into tf2's "cfg" folder. (Default location for this folder is "C:\Steam\steamapps\common\Team Fortress 2\tf\cfg")

2. Copy the "koi_custom" folder into tf2's "custom" folder. (Default location for this folder is "C:\Steam\steamapps\common\Team Fortress 2\tf\custom)

3. Add the "Advanced launch options" through steam (see below). These can also be found in autoexec.cfg and the gfx.cfg.
   // Advanced launch options (copy+paste)
   // -dxlevel 90 -sw -w 1920 -h 1080 -noborder -novid -useforcedmparms -noforcemaccel -noforcemspd
